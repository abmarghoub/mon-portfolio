## 1. Objectif du TP
Ce projet est un **portfolio personnel** développé avec **React + TypeScript**, stylisé avec **Tailwind CSS**, et utilisant les composants **shadcn/ui**.  
Il présente :
- Le profil personnel (nom, rôle, description)
- Projets réalisés
- Parcours professionnel (expériences)
- Formations et certifications
- Page contact
- Navigation SPA et SEO minimal
- Dark Mode toggle et intégration JSON-LD pour Google Knowledge Graph


---

## 2. Architecture du TP

### 2.1 Stack technologique

| Technologie          | Rôle dans le projet |
|---------------------|-------------------|
| **React 18**        | Framework front-end |
| **TypeScript**      | Typage statique et sécurité |
| **Vite**            | Bundler / dev server rapide |
| **Tailwind CSS v4** | Design et style utilitaire |
| **shadcn/ui**       | Composants UI réutilisables |
| **React Router DOM**| Gestion des routes SPA |
| **Framer Motion**   | Animations UI |
| **React Helmet Async** | SEO / gestion head |
| **ESLint + Prettier** | Qualité et format du code |
| **Vercel**          | Déploiement et hébergement |
| **EmailJS**          | Envoi de formulaires de contact par email |
